Главная

LuaApp это графический движок для языка программирования lua, написанный на golang с использованием gotk3 и gopher-lua. Особенность данного движка - это его кросс-платформенность.

На заметку: все функции виджетов пишутся через двоеточие кроме функции new. Пример:
btn = Button.new()
btn:setText("Text")
